# number-gessing-game
this is a simple guessing game written in python.
the programme generates a secret number from 1 to 20  and the player must guess it .
after each guess , the programme gives a hint until the correct number is found 
# How the game orks 
the programme generates a random number using random.randint().
the user enters a guess
the programme tells the user :" the  guess is bigger " or " the guess is smaller"
when the user finds ther correct number  ,the game ends
